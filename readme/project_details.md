* Short description
 * GeoGameHub is a Web platform for people who love gaming.

* What users can do
 * Users can register
 * Users can log in
 * Users can create, update and delete their posts.
 * Users can browse postfeed
 * Users can Search Posts
 * Users can like and comment on other users posts.

* Used technologies
 * Node.js / Express
 * Mongodb Atlas
 * Bcrypt
 * Mongoose
 * JWT
 * Joi
 * Body Parser
 * Escape html
 * Dotenv
 * Nodemon
 * Eslint
